# Country App with React-Redux & TypeScript

### [Demo of the country app](https://country-app-react-redux.netlify.app/)

## Installation

```
yarn install
```

## Running the app

```
yarn start
```

## Used tools & technologies

- React
- Redux
- Redux-saga
- React-router
- TypeScript
- CSS
- Material UI
- [Country API](https://restcountries.com/)
- Prettier
- ESLint
- Husky & lint-staged
